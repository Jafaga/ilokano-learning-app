# Ilokano Content Guide

## Content Workflow

1. Contributor drafts lesson content in the spreadsheet template.
2. Ilokano reviewer checks spelling, meaning, and cultural usage.
3. Approved content is added into the app.
4. Audio can be recorded after text is approved.

## Required Fields

Each exercise should include:

- Unit
- Lesson
- Exercise type
- Prompt
- Correct answer
- Wrong choices, if multiple choice
- Explanation
- Contributor name
- Reviewer name
- Status

## Status Labels

Use these labels:

```txt
Draft
Needs Review
Approved
Rejected
Published
```

## Lesson Design Tips

Keep early lessons short. A strong beginner lesson should have around 5 to 10 exercises.

Good lesson order:

1. Recognition: user sees Ilokano and chooses English.
2. Recall: user sees English and chooses Ilokano.
3. Typing: user translates a short word or phrase.
4. Review: user repeats the most important word again.

## Accuracy Note

All lesson content should be reviewed by fluent Ilokano speakers or qualified contributors before being treated as final. The current sample data is starter content and should be verified before public release.
